Welcome to mMOSS's documentation!
=================================

**mMOSS** is the **moderately Multiplayer Online Space Shooter**. **mMOSS** is
written in Python and uses the Twisted networking and Pygame game development
environment. 

**mMOSS** consists of client and server programs for implementing a 2D 
multiplayer space combat game in the style of Asteroids. A bare-bones trivial
mMOSS client, mmoss-ex.py, is included with mMOSS.
